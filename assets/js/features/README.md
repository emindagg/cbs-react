# ✨ Features - Special Features

Advanced and experimental features.

## 📂 Structure

```
features/
├── timeline/      - Temporal data visualization
├── astronomy/     - Astronomy globe (sun/moon position)
├── globe-view/    - 3D globe view
└── catalog/       - Data catalog integration
```

## 🎯 Purpose

Isolate special features that:
- Are optional/experimental
- Have complex dependencies
- May be disabled by users
- Need separate maintenance

## 📦 Feature Pattern

Each feature is self-contained:
- Independent initialization
- Own dependencies
- Can be disabled
- Documented separately
